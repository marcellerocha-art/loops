# 🎀 Loops - Exercícios de Lógica com Python

Bem-vindo a este repositório fofo de **exercícios de lógica com Python**! 🐍✨

## 📚 O que é este repositório?

Este é um espaço dedicado a praticar e dominar os conceitos fundamentais de loops (laços) em Python através de exercícios práticos e divertidos. Perfeito para quem está começando sua jornada na programação ou quer reforçar suas habilidades!

## 💫 Características

- ✨ Exercícios estruturados do básico ao avançado
- 🎯 Foco em compreensão de `for` e `while` loops
- 🐍 100% escrito em Python
- 💡 Soluções claras e bem comentadas
- 🌸 Uma comunidade fofa de aprendizado

## 🚀 Como usar

1. Clone este repositório:
```bash
git clone https://github.com/marcellerocha-art/loops.git
cd loops
```

2. Explore os exercícios e tente resolvê-los!
3. Compare suas soluções com as fornecidas
4. Aprenda e melhore cada vez mais! 🎉
